# Windchimes
That annoying jingling from next door, now in Minetest!  

### Recipes

Metal Windchime:  
```
W = default:wood
S = farming:string
I = default:steel_ingot
C = default:copper_ingot
+-+-+-+
|W|S|W|
+-+-+-+
|S|S|S|
+-+-+-+
|I|C|I|
+-+-+-+
```

Bamboo Windchime:  
```
W = default:wood
S = farming:string
J = default:junglewood
+-+-+-+
|J|S|J|
+-+-+-+
|S|S|S|
+-+-+-+
|J|W|J|
+-+-+-+
```
